 Student Stress Management System (OOP Project)
Description

This is a Java-based console application developed using Object-Oriented Programming (OOP) concepts.
The system helps track and analyze student stress levels based on daily activities such as study hours, sleep duration, and assignments.

It calculates a stress score, categorizes it into levels, and provides suggestions.
The application also stores data using file handling, allowing users to maintain a history of their records.

 Features
Add daily stress entries
Calculate stress score dynamically
Classify stress levels (Low / Medium / High)
Provide suggestions for stress management
Store data using file handling (persistent storage)
View all records
Filter records by date
Display statistics (average stress, distribution)
